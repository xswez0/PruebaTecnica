# PruebaTecnica
###
FrontEnd:
Proyecto en angular 13 hecho con visual studio code
- PruebaTecnicaUI
###
Backend:
Proyecto en visual studio 2022, usando NET6
- PruebaTenicaData
- PruebaTecnicaAPI
###
El backend usa entity framework core, se tiene que configurar la cadena de conexion, crear la base de datos y luego desde la consola de visual studio 2022, ingresar el update-database para que cree la tabla de usuarios.
